How to Use:

1. Upload an image with good contrast and clear features

2. Adjust the threshold to get a clear black and white pattern

3. Set the pattern size (16x16 is recommended for performance)

4. Click "Generate Pattern" to create the AR marker

5. Download the .patt file and use it in your AR.js project
